Рівень 2 
«Реалізуйте програму, яка визначає, чи є введене користувачем число простим»

#Рівень 2 завдання (3)
#отримуємо число від користувача
number = int(input("Введіть число: "))

#робимо перевірку та виведемо результат
if number <= 1:
    print("Це число не є простим")
else:
    is_prime = True

    for i in range(2, number):
        if number % i == 0:
            is_prime = False
            break

    if is_prime:
        print("Це просте число")
    else:
        print("Це число не є простим")

	Рівень 3
	«Створіть клас "Калькулятор" з методами для додавання, віднімання, множення та ділення. Виведіть результат обчислень для певного прикладу»

#Рівень 3 завдання (3)
class Calculator:

    #додавання
    def add(self, a, b):
        return a + b

    #віднімання
    def subtract(self, a, b):
        return a - b

    #множення
    def multiply(self, a, b):
        return a * b

    #ділення
    def divide(self, a, b):
        if b == 0:
            return "Помилка: ділення на нуль!"
        return a / b

#створюємо об'єкт класу
calc = Calculator()

#вхідні дані
a = int(input("Введіть перше число: "))
b = int(input("Введіть другек число: "))

# виводимо весь результат по діям
print("Додавання:", calc.add(a, b))
print("Віднімання:", calc.subtract(a, b))
print("Множення:", calc.multiply(a, b))
print("Ділення:", calc.divide(a, b))

Контрольні запитання
1.	Як оголосити змінну в Python?
Змінні визначаються просто шляхом присвоєння значення імені за допомогою оператора =

2.	Як вивести текст на екран у Python?
Для виведення тексту використовується функція print()

3.	Як отримати введення від користувача в Python?
Використовується функція input(), яка приймає рядок. Наприклад: name = input("Введіть ім'я")

4.	Які основні типи даних в Python?
Основними є: Числа (int, float, complex), Рядки (str), Списки (list), Кортежі (tuple), Словники (dict), Множини (set) та Булеві значення (bool)

5.	Як оголосити функцію в Python?
За допомогою ключового слова def, після якого йде назва функції та параметри в дужках
6.	Як зчитати змінну з клавіатури у Python?
Аналогічно до 3 питання, за допомогою input(). Якщо потрібно отримати саме число, результат input() слід обгорнути у функцію int() або float()

7.	Як створити умовний оператор у Python?
Використовуються конструкції if, elif та else

8.	Як створити цикл for у Python?
Використовується конструкція for variable in sequence:

9.	Як створити цикл while у Python?
Використовується ключове слово while, після якого йде умова

10.	Що таке клас та як його оголосити в Python?
Клас — це спосіб організації коду для створення об'єктів. Оголошується ключовим словом class
